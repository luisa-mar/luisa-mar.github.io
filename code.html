Auf dieser Seite werde ich versuchen Ihnen den Text, der die Website von Grund auf mit Hintergrundfunktionen und stilistischen Änderungen aufbaut, visuell näher zu bringen.

Als erstes möchte ich Sie darauf Aufmerksam machen, dass Sie auf jeder Website den Code sehen können, indem sie in der Menüleiste in Chrome auf "Anzeigen-Entwickler-Quelltext anzeigen" gehen oder in Safari unter "Entwickler-Webinformationen einblenden" klicken.

Elementeninformation

Mit der Elementauswahl können sie spezifische Elemente der Website anklicken und den HTML Code links sowie den css Code rechts unter "Stile" finden. Das war sehr wichtig für mich, um allfällige Fehler im Code zu finden.

Sie können die Elementenansicht auch auf die rechte Seite des Webfensters öffnen, damit sie den Code, neben der Website sehen.

Finder

Meine Website startete als erstes mit einem Tutorial: Create a New Blank Text File in Any Folder on Your Mac Mit diesem Tutorial schrieb ich meinen ersten Code für ein App, welches leere Textfiles im Finder erstellt.

Mit meinem neuen App konnte ich im Finder neue Files erstellen, welche ich mit den Endungen html, cssund js umbenannt habe. Bsp: index.html. Die Endungen stehen für die jeweilige Sprache in der der Code geschrieben wird.

Im HTML steht der gesamte Inhalt mit Texten, Bildern, Links etc. In CSS werden die Elemente aus dem HTML gestylt. Und Uu komplexe Funktionen auf Webseiten zu implementieren wird JavaScript eingesetzt, das ist fast jedes Mal, wenn eine Webseite mehr macht als nur dort sitzt und statische Informationen anzeigt.

Information um die Website zu starten

Die erste Linie Code sagt dem TextEditor, dass wir mit HTML Code schreiben werden. So werden vorprogrammierte "tags" wie "div" freigeschaltet.

Im "head" benennt man mit dem "title" tag die Website und verbindet mit dem "link" tag die CSS Seiten, welche diese spezifische HTML Seite stylen sollten. (rel=relation) Sonstige Links von zb. externen Fonts müssen ebenfalls im head stehen.

Navigationbar HTML

Für die Navigationbar erstellte ich eine Tabelle in der ich mit dem "a href=" tag Links zu den anderen Seiten einfügen konnte.

Alle Links sind in einer Class gruppiert, damit sie im CSS gleich gestylt werden.

Class="active" steht für die aktuell geöffnete Seite. Sie erhält im CSS einen farbigen Hintergrnud.

Navigationbar CSS

Ohne CSS würde die Navbar komplett anders aussehen. Im CSS habe ich die Platzierung geregelt, die Farbe, was geschieht, wenn man mit der Maus darüber fährt etc.

Index Body

Auf meiner Startseite habe ich ein Hintergrundbild gewählt, welches die ersten Zeilen Code aus meinem SublimeText-TextEditor zeigt. Es ist ebenfalls im "zhdk_website" Ordner unter einem neuen "img" Ordner abgelegt, damit mein HTML Programm dieses Bild findet.

Um Text über das Bild zu legen habe ich das Bild, sowie den Text in die selbe class gruppiert und den Text in eine weitere class gesetzt, damit ich ihn im CSS bearbeiten kann.

Ein "h" tag mit einer Nummer zwischen 1-6 steht für einen Titel, ein "p" tag für einen Text.
